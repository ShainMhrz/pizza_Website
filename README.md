# pizza_Website
///////////////////////////////////////////////////////////////////
to add any items on the menu list on the website just add :
  {
    "id": id,
    "name": "Name",
    "price": price,
    "image": "image.jpg/link"
  }
in product.Json file
//////////////////////////////////////////////////////////////////


/////////////////////////////////////////////////////////////////
note: don't forget "," before adding new {...}

e.g.
[

    "id": 11,

  },
{
    "id": 12,

    "image": "image.jpg/link"
  }
]
